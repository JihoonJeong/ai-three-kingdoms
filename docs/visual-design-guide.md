## 4. 캐릭터 비주얼 스펙

### 공통 규격
- **크기**: 512×512px (정방형)
- **포맷**: WebP (80% quality) — 약 50-80KB/장
- **배경**: 투명 (PNG로 생성 후 WebP 변환)
- **구도**: 흉상 (bust shot), 어깨~머리 위 공간 포함
- **조명**: 45° 상단 좌측 키 라이트, 은은한 림 라이트

### 표정 시스템 (Expression System)
각 장수 4가지 표정:

| 표정 코드 | 설명 | 사용 상황 |
|-----------|------|-----------|
| `default` | 기본 표정 | 평상시, 턴 시작 |
| `expression_1` | 장수별 고유 (분노/결의/사색 등) | 위기, 전투, 중요 결정 |
| `expression_2` | 장수별 고유 (기쁨/미소/자신감 등) | 성공, 승리, 좋은 소식 |
| `expression_3` | 장수별 고유 (걱정/경계/피로 등) | 실패, 위험, 불리한 상황 |

### 장수별 외형 키워드

#### 유비군 (蜀)
| 장수 | 핵심 키워드 | 표정 매핑 |
|------|-------------|-----------|
| 유비 | 인자한 얼굴, 쌍검, 녹색 도포, 유건(관모) | 기본/걱정/결의/미소 |
| 관우 | 붉은 얼굴, 장대한 수염, 청룡언월도, 녹색 전포 | 기본/분노/위엄/미소 |
| 장비 | 거친 얼굴, 번개 눈, 사모(투구), 장팔사모 | 기본/분노/포효/웃음 |
| 조자룡 | 젊은 미남, 은백 갑옷, 장창, 영웅적 | 기본/결의/미소/전투 |
| 제갈량 | 깃털 부채, 학창의(흰 도포), 지적, 도사풍 | 기본/사색/미소/경고 |
| 황충 | 백발 노장, 강궁, 갑옷, 근엄 | 기본/결의/분노/피로 |
| 위연 | 험상궂은 얼굴, 반골상, 투구, 야심적 | 기본/야심/분노/불만 |
| 간옹 | 문관 복식, 온화한 얼굴, 두루마리 | 기본/미소/걱정/사색 |

#### 조조군 (魏)
| 장수 | 핵심 키워드 | 표정 매핑 |
|------|-------------|-----------|
| 조조 | 날카로운 눈, 짧은 수염, 검은 갑옷, 왕관 | 기본/냉소/분노/오만 |
| 하후연 | 건장, 경장갑, 기마, 창 | 기본/돌격/분노/경계 |
| 조인 | 충실한 인상, 중갑, 방패, 수성 전문 | 기본/수성/결의/경계 |
| 채모 | 형주 관리 복식, 교활한 눈 | 기본/불안/교활/공포 |
| 장윤 | 수군 복식, 평범한 인상 | 기본/불안/결의/공포 |

#### 손권군 (吳)

## 5. 맵 디자인 스펙

### 전략 맵
- **크기**: 1920×1080px (16:9)
- **스타일**: 수묵화 일러스트 — 장강 유역, 산수화 구도
- **핵심 요소**: 장강(長江) 중심, 5개 도시 위치, 적벽 전장
- **인터랙티브 오버레이**: SVG로 도시 마커·군대 이동선·전투 표시

### 도시 마커
- **규격**: 64×64px SVG
- **스타일**: 세력별 색상의 성곽 아이콘 (수묵화 터치)
- **상태**: 기본 / 선택됨 (발광) / 전투 중 (진동)

### 지리적 배치 (대략적 좌표)
```
장릉 (25%, 20%)  ─── 남군 (50%, 25%)
                        │
                  적벽 (55%, 55%)  ─── 강하 (75%, 35%)
                                       │
                                  하구 (80%, 60%)
                                       │
                                  시상 (85%, 80%)
```

## 6. 전투 화면 스펙

### 레이아웃 구조
```
┌─────────────────────────────────────┐
│ 전투 제목 바                         │
├───────────┬─────────┬───────────────┤
│ 아군 장수  │ 전투 배경│ 적군 장수     │
│ 초상화    │ (전체)  │ 초상화        │
├───────────┴─────────┴───────────────┤
│ 아군 HP/사기 바     적군 HP/사기 바  │
├─────────────────────────────────────┤
│ 전술 카드 선택 영역                  │
├─────────────────────────────────────┤
│ 전투 로그                           │
└─────────────────────────────────────┘
```

### 전투 배경
| 지형 | 설명 | 분위기 |
|------|------|--------|
| 수상전 | 장강 위 전함, 파도, 안개 | 긴장감, 웅장함 |
| 평야 | 넓은 벌판, 군기, 먼지 | 개활, 돌격감 |
| 성곽 | 성벽, 사다리, 불화살 | 공방, 긴박함 |

### 전술 카드
- **규격**: 120×160px
- **구성**: 아이콘(상단) + 전술명(중앙) + 간략 효과(하단)
- **스타일**: 수묵화 프레임, 금색 테두리 (선택 가능 시)
- **비활성**: 회색 처리 + 자물쇠 아이콘

## 7. 이벤트 컷신 스펙

### 규격
- **전체 화면 일러스트**: 1920×1080px
- **텍스트 오버레이**: 하단 20% 영역, 반투명 검정 배경
- **캐릭터 사이드 팝업**: 256×256px 초상화

### 연출 시퀀스
1. 화면 페이드 인 (0.5초)
2. 일러스트 표시 (약간의 Ken Burns 효과 — 느린 줌)
3. 하단 텍스트 타이핑 (20ms/글자)
4. 관련 장수 초상화 사이드 슬라이드 인
5. 유저 클릭 또는 3초 후 자동 진행
6. 페이드 아웃 (0.5초)

## 8. UI 프레임 스펙

### 메인 게임 화면 레이아웃
```
┌──────────────────────────────────────────┐
│ 상단 바: 턴 정보 | 자원 | 행동 포인트      │
├────────────┬─────────────────────────────┤
│            │                             │
│ 좌측 패널  │       메인 영역              │
│ (장수 목록 │   (전략 맵 / 도시 / 전투)    │
│  또는      │                             │
│  도시 정보)│                             │
│            │                             │
├────────────┴─────────────────────────────┤
│ 하단: 행동 버튼 | 전투 로그 | 제갈량 조언  │
└──────────────────────────────────────────┘
```

### UI 컴포넌트 스타일
- **패널 배경**: 한지 텍스처 (`#f5f0e8`) + 수묵화 테두리 (2px `#1a1a2e`)
- **버튼**: 둥근 모서리 없음 (직각), 붓 획 테두리, 호버 시 발광
- **아이콘**: 최소한의 선화 스타일, 세력 색상 반영
- **진행 바**: 수묵화 붓 획 모양, 채워진 영역은 세력 색상
- **드롭 섀도우**: 없음 (수묵화 스타일에 맞지 않음)
- **모서리**: 직각 또는 미세한 라운드 (2px max)

## 9. 애니메이션 가이드

| 요소 | 애니메이션 | 속도 | 이징 |
|------|-----------|------|------|
| 패널 전환 | 슬라이드 인 | 300ms | ease-out |
| 초상화 등장 | 페이드 인 | 400ms | ease |
| 전술 선택 | 스케일 업 (1.05x) | 200ms | ease-out |
| 피해 표시 | 셰이크 + 플래시 | 400ms | ease-in-out |
| 이벤트 텍스트 | 타이핑 | 20ms/char | linear |
| 맵 군대 이동 | 패스 드로우 | 1000ms | ease-in-out |
| 턴 전환 | 페이드 through black | 600ms | ease |

## 10. 에셋 생성 도구

프롬프트 시트(`docs/asset-prompt-sheet.md`)의 프롬프트를 아래 도구에서 사용:

| 도구 | 용도 | 비고 |
|------|------|------|
| **Google Flow** | 캐릭터 초상화, 이벤트 일러스트 | 일관된 스타일 유지에 강점 |
| **Google Veo** | 전투 이펙트 영상 클립 (화공, 동남풍) | 짧은 클립 (3-5초) |
| **Gemini** | 프롬프트 리파인, 스타일 일관성 체크 | 보조 도구 |

### 에셋 후처리 체크리스트
- [ ] 배경 제거 (캐릭터 초상화)
- [ ] 512×512 리사이즈 (초상화) / 1920×1080 (배경)
- [ ] WebP 변환 (80% quality)
- [ ] 파일명 규칙 준수 (`{generalId}/{expression}.webp`)
- [ ] 세력별 톤 일관성 검수
